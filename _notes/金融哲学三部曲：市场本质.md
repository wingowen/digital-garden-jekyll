本书提供了一个较为新颖的视角：丢掉数学模型，回归到人类本身来研究市场，以社会学的角度来看待思考市场。

**第一部分：雏形**

作者认为市场是人类社会的一部份，它不是被创造出来的，而是随着人类社会的进步演化出来的，

作者通过自然界中的灵长类动物的行为来印证这一说法。

作者提到了一个有趣的概念，叫做“全雄群”。

什么是全雄群？全雄群是灵长类动物的”政治活动“中一种特殊的结盟形式：由竞争群体领袖失败的雄性、刚刚性成熟的年轻雄性、从周围其它群体流浪而来的雄性等构成的全雄群。作者认为“全雄群”是市场的早期形态，是自由联合的雏形。这种联合无非是为了维护“个体自由”以及“社会地位”。

市场不该由经济学描述的供求关系和随机过程来决定，而是在“个体自由”以及“社会地位”这两个关键字下演化诞生的。

黑猩猩中的“全雄群”。

巴诺布猿中的“全雄群”。

分别对应着两种市场形态，现实中两种形态都是存在的，说明偏好的发展不是非黑即白，一切都是为了生存而服务，这使得人类对社会的变化具有很强的适应性。

**第二部分：人性**

传统的经济学研究总是会理性的研究市场，而忽略了市场的参与者，人的人性。

人寻求结盟，在市场上，人们通过结盟来实现共同的目标。这种结盟可以是非正式的，如信任和声誉的建立，也可以是正式的，如合同和法律协议。结盟有助于降低交易成本，提高市场效率。

人维护等级，等级可以影响市场动态，包括定价权和资源分配。

人追求自由，市场提供了一种自由交换的机制，允许个体自由地买卖商品和服务。然而，自由并不意味着无序，市场秩序是通过法律、规则和社会规范来维持的。自由是市场的核心价值之一，但它需要在一定的框架内运作。

**第三部分：演化**

市场的的成熟与有效，背后是人愈发的独立与自由。

在市场经济中，面子和关系可以转化为经济资本，影响交易和合作的方式。

在市场演化的过程中，礼物交换的习俗逐渐被商品交易的逻辑所取代，反映了从部落社会到市场经济的转变。

随着个体的独立性和自由度提高，市场机制也更加成熟和有效，个体能够在市场中自由地买卖商品和服务。

资产的积累和货币的使用促进了市场的发展，使得市场能够更有效地分配资源和进行交易。

**第四部分：机制**

保护自由，就是捍卫市场机制。

不同市场形态如何影响交易的方式和效率。

市场的基本功能，如价格发现、资源配置和风险管理。他强调了市场在激发创新和提高生产效率方面的作用。
 
市场治理的机制，包括法律法规、行业标准和企业内部治理。他指出，良好的市场治理对于保护消费者权益和维护市场秩序至关重要。

资产估值的原则和方法。他强调了估值在资本市场中的重要性，以及它如何影响投资决策和市场流动性。

理解市场的本质需要深入探讨人性，因为市场是人类社会的一部分，反映了人类的价值观和行为模式。市场不仅仅是一个交易的场所，它是一种社会秩序，这种秩序是建立在人类对自由、合作和社会结构的需求之上的。

# 雏形

> ***==市场是自由人的联合==***

**大自然母亲**

市场不是人类独有的，大自然母亲在每个物种的基因里都刻上市场，并在漫长的进化中演进市场。

对于优势物种，大自然提供了提供了它们生存的一切资源，它们会繁衍更多后代；对于好公司和好项目，市场给它们提供了便宜的融资和广阔的发展机遇。

**全雄群视角**

灵长类动物的”政治活动“中一种特殊的结盟形式：由竞争群体领袖失败的雄性、刚刚性成熟的年轻雄性、从周围其它群体流浪而来的雄性等构成的全雄群。

类比：人类的梁山好汉、公务员下海找机会。

个体获得了更大的发展空间，不再受制于原来的群体（五月花）；个体经历了磨难后，回归原有部落获得更高的地位（梁山好汉）。个体追求更大的生存空间（商品市场），个体追求更高的社会地位（资本市场）。

市场效率源于捍卫个人自由。

看待市场的最佳角度是全雄群，而不是供求关系和随机过程。全雄群只有两个关键字：个体自由（拥有更多的社会资料，建立商品市场）和社会地位（更公平有序的社会秩序，建立资本市场）。

自由与秩序的对立统一：自由需要秩序保障，秩序需要自由捍卫。

1. **大自然母亲**：周洛华从自然界的角度出发，探讨了市场机制如何与自然界的生存法则相联系，以及人类如何从自然界中学到了交换和合作的概念。
    
2. **全雄群视角**：作者通过对灵长类动物社会行为的观察，尤其是全雄群的现象，来类比市场机制的早期形态。全雄群是年轻雄性动物在成熟后离开原生群体，形成临时的全雄群，这反映了一种自由联合的雏形。
    
3. **黑猩猩的“政治”**：周洛华分析了黑猩猩的社会结构和“政治”行为，指出黑猩猩群体中的权力斗争和等级制度，与人类社会中的市场机制形成对比。
    
4. **巴诺布猿的“政治”**：与黑猩猩相对，巴诺布猿的社会结构更加平等和和谐。作者通过比较这两种灵长类动物，来探讨市场机制如何在人类社会中形成，以及它与人类对自由和社会秩序的需求之间的关系。

市场的起源不仅仅是一个经济过程，而是一个深植于人类本性和社会行为中的复杂现象。市场的雏形可以在自然界、人类社会的早期形态以及灵长类动物的行为中找到踪迹

# 人性

> ==***市场是自由社会的秩序***==

**理性**

无法脱离人性谈市场，纯粹的理性并不适用于市场。

传统经济学假设市场参与者是理性的，总是追求自身利益最大化。然而，周洛华指出，市场参与者的行为并不总是完全理性的，因为他们受到情感、习惯、社会规范等多种因素的影响。市场行为是复杂的，不能简单地用理性来解释。

**结盟**

在市场上，人们通过结盟来实现共同的目标。这种结盟可以是非正式的，如信任和声誉的建立，也可以是正式的，如合同和法律协议。结盟有助于降低交易成本，提高市场效率。

**等级**

虽然市场倾向于促进平等和自由，但等级制度在市场中仍然存在。例如，公司内部有管理层和员工的等级结构，市场中有大企业和小企业之间的权力差异。等级可以影响市场动态，包括定价权和资源分配。

**自由**

市场提供了一种自由交换的机制，允许个体自由地买卖商品和服务。然而，自由并不意味着无序，市场秩序是通过法律、规则和社会规范来维持的。自由是市场的核心价值之一，但它需要在一定的框架内运作。

1. **理性**：周洛华讨论了理性在市场中的作用，指出市场参与者的决策并非总是基于完全理性的计算，而是受到情感、习惯、社会规范等多种因素的影响。他强调了理性的局限性，并探讨了如何在一个不完全理性的世界里理解市场行为。
    
2. **结盟**：作者分析了人们在市场中如何通过结盟来实现共同的目标。这种结盟可以是非正式的，如信任和声誉的建立，也可以是正式的，如合同和法律协议。结盟有助于降低交易成本，提高市场效率。
    
3. **等级**：周洛华探讨了等级制度在市场中的存在，尽管市场倾向于促进平等和自由，但等级制度在市场中仍然存在。他讨论了等级如何影响市场动态，包括定价权和资源分配。
    
4. **自由**：作者强调了自由在市场中的核心地位，认为市场的成熟和有效性与人的独立性和自由度密切相关。他探讨了个体的自由度如何影响市场机制的运作。
# 演化

> *==**人变得越独立和越自由，市场就变得越成熟和越有效**==*

1. **面子与关系**：周洛华分析了面子文化和关系网络在市场演化中的作用。他指出，在市场经济中，面子和关系可以转化为经济资本，影响交易和合作的方式。
    
2. **礼物与商品**：作者探讨了礼物交换与商品交易之间的关系。在市场演化的过程中，礼物交换的习俗逐渐被商品交易的逻辑所取代，反映了从部落社会到市场经济的转变。
    
3. **部落与个人**：周洛华讨论了从部落社会到个体化社会的转变。随着个体的独立性和自由度提高，市场机制也更加成熟和有效，个体能够在市场中自由地买卖商品和服务。
    
4. **资产与货币**：作者分析了资产和货币在市场演化中的重要性。资产的积累和货币的使用促进了市场的发展，使得市场能够更有效地分配资源和进行交易。

市场不仅仅是一个经济机制，而是人类社会的一种秩序。市场的成熟和有效性与人的独立性和自由度密切相关。随着人们追求更大的自由和独立，市场机制也相应地发展和完善。

# 机制

> ***==保护自由就是捍卫市场机制==***

1. **形态**：这一部分讨论了市场的各种形态，包括传统的市场和现代的电子市场。作者分析了不同市场形态如何影响交易的方式和效率。
    
2. **功能**：周洛华探讨了市场的基本功能，如价格发现、资源配置和风险管理。他强调了市场在激发创新和提高生产效率方面的作用。
    
3. **治理**：在这部分，作者讨论了市场治理的机制，包括法律法规、行业标准和企业内部治理。他指出，良好的市场治理对于保护消费者权益和维护市场秩序至关重要。
    
4. **估值**：估值是市场机制中的一个重要环节，作者分析了资产估值的原则和方法。他强调了估值在资本市场中的重要性，以及它如何影响投资决策和市场流动性。
